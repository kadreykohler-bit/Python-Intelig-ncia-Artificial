# 🐍 Desafios de Código - Python Fundamentals (DIO)

Este repositório registra minha evolução técnica na trilha de Python da [DIO](https://www.dio.me/). Aqui, apliquei conceitos fundamentais para resolver problemas de lógica e manipulação de dados.

## ✍️ O que foi aprendido em cada desafio:

### 1. Par ou Ímpar 🔢
**Conceito chave:** Operadores Aritméticos e Expressões Condicionais.
* Aprendi a utilizar o operador de módulo (`%`) para verificar o resto de uma divisão.
* Implementei o **If Ternário** para escrever um código mais limpo e direto em uma única linha.

### 2. Verificador de Ano Bissexto 📅
**Conceito chave:** Lógica Booleana e Funções.
* Pratiquei a construção de condições complexas usando os operadores lógicos `and`, `or` e `!=`.
* Aprendi a importância da **identação** (espaçamento) correta para definir o escopo de uma função no Python.

### 3. Contador de Vogais 🗣️
**Conceito chave:** Iteração (Loops) e Manipulação de Strings.
* Aprendi a percorrer cada caractere de uma string usando o laço `for`.
* Utilizei o operador `in` para verificar se um elemento pertence a um conjunto de dados (vogais maiúsculas e minúsculas).

## 🚀 Tecnologias e Ferramentas
* **Linguagem:** Python 3.x 🐍
* **Editor:** VS Code com extensões de Linter.
* **Controle de Versão:** Git e GitHub para versionamento de código.
