# 🎓 Formação Python Fundamentals - DIO

Este repositório contém o registro da minha jornada na **Formação Python Fundamentals** da [DIO](https://web.dio.me/). A trilha percorreu desde a sintaxe básica até conceitos avançados de engenharia de software com Python. 🐍

## 📚 Estrutura da Formação

A formação foi dividida em 4 módulos principais, totalizando 19 cursos e 3 desafios práticos de código:

### 1. Fundamentos do Python 🧱
Foco em lógica inicial, tipos de dados e estruturas de controle.
* **Cursos:** Ambiente de desenvolvimento, operadores, estruturas condicionais/repetição e manipulação de strings.
* **Desafio Prático:** Explorando Fundamentos de Python (Verificação de Paridade e Ano Bissexto).

### 2. Estruturas de Dados 📊
Aprendizado sobre como organizar e manipular informações de forma eficiente.
* **Cursos:** Listas, Tuplas, Conjuntos, Dicionários e Domínio de Funções.
* **Desafio Prático:** Dominando Estrutura de Dados com Python (Contagem de Vogais).

### 3. Programação Orientada a Objetos (POO) 🏛️
Introdução ao paradigma de objetos para criação de sistemas robustos.
* **Cursos:** Classes, Herança, Encapsulamento, Polimorfismo e Classes Abstratas.
* **Desafio Prático:** Treinando Orientação a Objetos com Python.

### 4. Tópicos Avançados 🚀
Recursos modernos e manipulação de arquivos.
* **Cursos:** Decoradores, Iteradores, Geradores, Data/Hora e I/O de arquivos.

## 🛠️ Desafios de Código em Destaque

1.  **Verificador de Ano Bissexto:** Implementação de lógica condicional para validar regras do calendário gregoriano.
2.  **Contador de Vogais:** Algoritmo que utiliza iteração e conjuntos para análise de strings.
3.  **Verificador de Paridade:** Uso de operadores aritméticos para tomada de decisão.
