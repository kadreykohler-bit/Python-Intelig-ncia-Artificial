# 🐍 Desafios de Código - Python Fundamentals (DIO)

Este repositório contém as soluções para os desafios básicos de lógica e sintaxe desenvolvidos durante a trilha **Python Fundamentals**.

## 🚀 Desafios Resolvidos

### 1. Verificador de Paridade 🔢
Script que identifica se um número inteiro é par ou ímpar de forma eficiente.
* **Conceito Chave:** Uso do operador de módulo (`%`) para verificar o resto da divisão.
* **Destaque:** Implementação de uma estrutura condicional simplificada para exibição do resultado.

### 2. Verificador de Ano Bissexto 📅
Algoritmo que valida se um ano específico atende aos critérios do calendário gregoriano.
* **Lógica Aplicada:** * O ano deve ser divisível por 400 **OU**
    * Ser divisível por 4 e **não** divisível por 100 ao mesmo tempo.
* **Função:** Encapsulado na função `verificador_ano_bissexto()`.

### 3. Contagem de Vogais 🗣️
Um contador inteligente que analisa strings e identifica a presença de vogais, ignorando a diferença entre maiúsculas e minúsculas.
* **Funcionamento:** Define um conjunto padrão (`"aeiouAEIOU"`) e utiliza um laço de repetição `for` para iterar sobre o texto.
* **Contador:** Incrementa uma variável a cada correspondência encontrada.
