# Desafio: Verificação de Paridade (Par ou Ímpar)

Este projeto contém uma solução em Python para o desafio de identificar se um número inteiro fornecido pelo usuário é par ou ímpar.

## 📖 Descrição
O objetivo é receber um número inteiro como entrada e retornar uma string indicando se o número é `Par` ou `Ímpar`.

### Regras
- Se o número for divisível por 2 (resto da divisão igual a 0), o resultado é **Par**.
- Caso contrário, o resultado é **Ímpar**.

## 🛠️ Tecnologias Utilizadas
- Python 3
