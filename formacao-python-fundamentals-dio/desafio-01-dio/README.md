# 🔢 Desafio Ímpar 01

Projeto simples desenvolvido em Python com foco em lógica de programação.

🎯 Objetivo: praticar estruturas condicionais, operadores matemáticos e boas práticas iniciais em Python.

---

## 🚀 Descrição

Este projeto verifica se um número informado é **ímpar ou par**, aplicando conceitos fundamentais como:

- Entrada de dados
- Operadores aritméticos (% módulo)
- Estruturas condicionais (if / else)
- Organização básica de código

---

## 🛠️ Tecnologias Utilizadas

- Python
- VS Code
- Git & GitHub
