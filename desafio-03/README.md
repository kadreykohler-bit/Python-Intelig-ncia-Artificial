# 🐍 Treinando POO com Python

Este repositório contém resoluções de desafios da **DIO** focado em Orientação a Objetos.

## 🛠️ Tecnologias Utilizadas
* Python 3.10+
* Boas práticas de estilo (PEP 8)

## 📖 Aprendizados Principais
> "A classe é o molde, o objeto é a criação."

Durante os desafios de `Calculadora`, `Pessoa` e `Conversor de Temperatura`, dominei:
1. **Instanciação:** Como criar objetos a partir de classes.
2. **Encapsulamento Simples:** Organizar lógica de negócio dentro de métodos específicos.
3. **Robustez:** Usar `try-except` para evitar que o código quebre com entradas erradas.
