# 🐍 Desafios de Processamento de Dados com Python

Este repositório contém uma coleção de scripts práticos desenvolvidos para resolver problemas comuns de tratamento de dados, como formatação de datas, conversão de unidades e manipulação de strings.

## 🧠 Aprendizados Técnicos

Ao desenvolver estes desafios, apliquei conceitos fundamentais de Python:

* **Manipulação de Strings**: Uso de `.split()` para fatiar dados e `.strip()` para limpeza de espaços. ✂️
* **Tratamento de Listas**: Criação de listas, iteração com `for` e uso de `.append()`. 📦
* **Conversão de Tipos (Casting)**: Transformação de strings em `float` para cálculos precisos. 🔢
* **Formatação Dinâmica**: Uso de **f-strings** para construir textos organizados. 🏗️

---

## 🛠️ Projetos Incluídos

### 1. Conversão de Temperatura (Celsius -> Fahrenheit) 🌡️
Este script recebe uma lista de temperaturas em Celsius e as converte para a escala Fahrenheit.
- **Fórmula**: $TF = (TC \times 9/5) + 32$

### 2. Formatação de Datas (BR -> Internacional) 📅
Converte datas no formato brasileiro `DD-MM-YYYY` para o formato internacional `YYYY/MM/DD`, comum em bancos de dados.

### 3. Extração de Domínios de E-mail 📧
Analisa uma lista de e-mails e extrai apenas o domínio de cada endereço (ex: @gmail.com).
